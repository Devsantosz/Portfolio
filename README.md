## Sobre o projeto

Portfólio pessoal criado para apresentar minha trajetória como desenvolvedor web. O design é minimalista, com identidade visual em preto, roxo e amarelo, construído do zero sem frameworks ou bibliotecas de UI — apenas HTML, CSS e JavaScript puro.

O objetivo é atender tanto **recrutadores** (que buscam clareza sobre skills e experiência) quanto **clientes** (que precisam entender o que pode ser entregue e como entrar em contato).

## Seções

| Seção | Descrição |
|---|---|
| **Hero** | Apresentação com proposta de valor, botão para projetos e download de currículo |
| **Sobre** | Foto, bio, diferenciais e links sociais |
| **Vídeo** | Apresentação em vídeo para criar conexão mais pessoal |
| **Skills** | Tecnologias organizadas por categoria (Frontend, Backend, Ferramentas) |
| **Formação** | Timeline com cursos e ensino médio em ordem cronológica |
| **Projetos** | Grid de 9 projetos com imagem, descrição, stack e link |
| **Contato** | Formulário + links do LinkedIn e GitHub |

---

## Tecnologias

- **HTML5** — estrutura semântica e acessível
- **CSS3** — variáveis CSS, Grid, Flexbox, animações e responsividade completa sem media queries externas
- **JavaScript** — menu mobile, loader animado e interações
- **Font Awesome** — ícones
- **Google Fonts** — `Syne` (títulos) + `DM Mono` (labels e código)
- **Vercel** — deploy contínuo via GitHub

## Como rodar localmente

Não há dependências ou build step. Basta clonar e abrir no navegador.

```bash
# Clone o repositório
git clone https://github.com/Devsantosz/portfolio.git

# Entre na pasta
cd portfolio

# Abra no navegador (ou use Live Server no VS Code)
open index.html
```

> **Recomendado:** use a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) no VS Code para hot reload durante o desenvolvimento.
