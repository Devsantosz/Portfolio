Minhas habilidades:

HTML5

HTML é a linguagem usada para criar páginas de sites. Ele organiza o conteúdo (textos, imagens, links) usando "tags", que dizem ao navegador como exibir cada parte da página.

CSS3

CSS é uma linguagem que define a aparência de um site. Ela permite mudar cores, fontes, tamanhos e o layout dos elementos, deixando as páginas mais bonitas e organizadas.

JAVASCRIPT

JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web. Ele permite que você adicione interatividade e dinamismo às páginas da web.
